# 🏥 Система управления медицинским центром

![.NET](https://img.shields.io/badge/.NET-8.0-512BD4?style=flat-square&logo=dotnet)
![Windows Forms](https://img.shields.io/badge/Windows_Forms-0078D7?style=flat-square&logo=windows)
![C#](https://img.shields.io/badge/Language-C%23-239120?style=flat-square&logo=c-sharp)

**Медицинский центр** — это современное десктопное приложение для автоматизации работы регистратуры и врачей в частных медицинских клиниках. Приложение разработано на **C#** (WinForms) и использует локальную файловую систему (JSON) в качестве базы данных, что позволяет ему работать автономно без необходимости настройки тяжеловесных СУБД (SQL). 

---

## ✨ Ключевые возможности

* 🧑‍⚕️ **Управление врачами**: Добавление, редактирование информации о докторах (ФИО, специализация, контактные данные). Проверка на наличие будущих приемов при попытке удаления.
* 👥 **Учет пациентов**: Детальная карточка пациента (ФИО, телефон, адрес, дата рождения, страховой полис).
* 📅 **Расписание и бронирование**: Запись на прием через удобный интерфейс с календарем. Автоматическая защита от записи разных пациентов на одно и то же время.
* 📊 **Отчетность и экспорт**: Генерация статистики по популярности врачей и списку посещений с возможностью выгрузки в красиво оформленный **HTML-отчет**.
* 🛡️ **Ролевая система безопасности**: Разграничение прав доступа (Администратор имеет полный доступ, врач — ограниченный).

## 🛠️ Техническая реализация (Архитектура)

Проект спроектирован с учетом принципов современного ООП и концепции Чистой Архитектуры:
- **Абстрактный класс** `BaseEntity` служит основой для всех моделей данных, инкапсулируя сквозные параметры (Id и Дата создания).
- **Интерфейсы**: Проект слабо связан и зависит от абстракций (`IRepository<T>`, `IAuthService`, `IValidator`).
- **Data Access Layer (DAL)**: Данные обрабатываются через generic-класс `JsonRepository<T>`, который автоматически сериализует и десериализует C#-классы в JSON-файлы, расположенные в папке `/Data`.

## 🚀 Как запустить проект

1. Склонируйте репозиторий на свой компьютер:
   ```bash
   git clone https://github.com/Kasper215/Medical_Center.git
   ```
2. Откройте терминал в папке проекта и запустите сборку:
   ```bash
   dotnet run --project MedCenterApp.csproj
   ```
   *(Либо откройте `MedCenterCourse.sln` в Visual Studio / Rider и нажмите кнопку Run).*

### Демонстрационный вход
При первом запуске программа сгенерирует базу данных (папку `Data`). Для входа в систему используйте:
